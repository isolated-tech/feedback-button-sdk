Pullreque.st Feedback Button Test

Test the feedback button! Look for the floating "Feedback" button in the bottom-right corner.

How it works

This page demonstrates the Pullreque.st feedback widget. The button is initialized with this code:

<script src="https://pullreque.st/cdn/button.js"></script>
<script>
  PullrequeStButton.init({
    projectKey: 'prj_pk_3V4nxbheZC3RHLJYRuXtkx26jLA351vtZ'
  })
</script>

Customization Options

You can customize the button appearance and position:

PullrequeStButton.init({
  projectKey: 'your_project_key',
  position: 'bottom-right',  // or 'bottom-left', 'top-right', 'top-left'
  buttonText: 'Feedback',     // Custom button text
  primaryColor: '#000000',    // Custom color (hex)
  endpoint: 'http://localhost:3000/api/feedback'  // For local development
})
